# ixland-wasm

This directory is reserved for future engine-neutral WebAssembly boundaries.

## Structure

This directory contains:
- `ixland-wasm-engine/` - Engine abstraction layer
- `ixland-wasm-host/` - Host contract between guest runtimes and iXland
- `ixland-wasm-wasi/` - WASI guest ABI policy and conformance

## Compiled Artifacts

Compiled Wasm artifacts are a future concern of `ixland-packages`.
