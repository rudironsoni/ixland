# IXLand Python Wheel Index

Custom Python wheel index for IXLand.

## Overview

This directory contains wheel metadata and scripts for managing Python package distribution for the IXLand environment.

## iOS Platform Tags

IXLand uses custom platform tags for iOS:

```
ixland_310_ios_arm64
```

## Contents

- Wheel metadata files
- Platform tag definitions
- Distribution scripts
